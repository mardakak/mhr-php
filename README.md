# mhr-php

Follow the [mhr-cfw](https://github.com/denuitt1/mhr-cfw/tree/main) instructions, but use worker.php instead of a Cloudflare Worker. Don't forget to change `example.com` to your own domain before using it in the Google Apps Script code.

تمام مراحل راهنمای [mhr-cfw](https://github.com/denuitt1/mhr-cfw/tree/main) را دنبال کنید، اما به جای Cloudflare Worker از worker.php استفاده کنید. فراموش نکنید که قبل از قرار دادن کد در Google Apps Script، دامنه خودتان را جایگزین `example.com` کنید.
